# Server List Switcher

[![Discord](https://img.shields.io/badge/Discord-7289DA?style=for-the-badge&logo=discord&logoColor=white)](https://discord.gg/tVWXETy6GZ)
[![Modrinth](https://img.shields.io/badge/Modrinth-00AF5C?style=for-the-badge&logo=modrinth&logoColor=white)](https://modrinth.com/mod/serverlist-switcher)

Server List Switcher is a comprehensive Fabric mod that revolutionizes how you organize your multiplayer servers. Create custom categories, assign servers to multiple groups, and switch between them seamlessly - perfect for organizing PVP, SMP, creative, and other server types.

## ✨ Key Features

### 🗂️ **Smart Organization**
- **Custom Categories**: Create unlimited categories to organize your servers
- **Multi-category Assignment**: Assign servers to multiple categories simultaneously  
- **"All" View**: Quick access to see all your servers at once
- **Persistent Storage**: Your categories and assignments are saved automatically

### 🎮 **Intuitive Interface**
- **Quick Add Button**: + button in top-right to create new categories instantly
- **Arrow Navigation**: Use ◄ ► arrows or keyboard arrow keys to switch categories
- **Hover Actions**: Hover over any server to reveal a + button for category assignment
- **Visual Feedback**: Clean, modern interface with helpful tooltips

### 🌍 **Multilanguage Support**
- **9 Languages**: English, German, French, Spanish, Italian, Portuguese, Russian, Chinese, Japanese
- **Auto-detection**: Automatically detects your system language
- **Easy Switching**: 🌐 button for quick language changes

### 🔧 **Cross-Version Compatibility**
- **Full Support**: Works with Minecraft 1.18 through 1.21.11
- **Version Optimization**: Optimized for each Minecraft version
- **Consistent Experience**: Same features across all supported versions

## 🎯 **Perfect For**

- **PVP Enthusiasts**: Separate competitive servers from casual gameplay
- **Multi-server Players**: Organize different communities and friend groups  
- **Content Creators**: Categorize servers for different content types
- **Server Admins**: Manage testing and production environments separately

## 🛠️ **Installation**

1. **Download** the latest version from Modrinth
2. **Install** Fabric Loader for your Minecraft version
3. **Place** the mod JAR in your `mods` folder
4. **Launch** Minecraft and enjoy organized server lists!
